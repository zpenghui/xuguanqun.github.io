5个 JSON.stringify() 的秘密（译文）

  • 译文
作者: 徐冠群

目录

  1. 1. 第二个参数（数组）
  2. 2. 第二个参数（函数）
  3. 3. 第三个参数（Number）
  4. 4. 第三个参数（String）
  5. 5. toJSON 函数

正文

JSON.stringify() 函数可以将 JavaScript中的对象或值转换成JSON字符串。

作为一个JavaScript开发者，JSON.stringify() 是调试时最常用的函数，难道我们不能用我们的朋友 console.log() 来做同样的事吗？我们来试试。

//初始化一个对象
const user = {
 "name" : "Prateek Singh",
 "age" : 26
}
console.log(user);
// [object Object]

哎呀！console.log() 无法帮助我们打印出所需的结果，它打印了 [object Object] ，因为当对象转换成字符串时默认为 [object Object] 。所以我们要先用 JSON.stringify() 将对象转换为字符串，然后在控制台中打印，如下所示。

const user = {
 "name" : “Prateek Singh”,
 "age" : 26
}
console.log(JSON.stringify(user));
// "{ "name" : "Prateek Singh", "age" : 26 }"

一般情况，开发人员使用这个stringify函数的方式很简单，就像我们在上面所做的那样。但我要告诉你一些隐藏的秘密，这些小秘密会让你的生活变得轻松。

# 1. 第二个参数（数组）

是的，我们的 stringify函数 也可以有第二个参数。它是要在控制台中打印的对象的键数组。看起来很简单？让我们仔细看看。我们有一个对象产品和我们想知道产品的名称。当我们将其打印为：

console.log(JSON.stringify(product));

它给出以下结果

{"id"}:"0001","type":"donut","name":"Cake","ppu":0.55,"batters":{"batter":[{"id":"1001","type":"Regular"},{"id":"1002","type":"Chocolate"},{"id":"1003","type":"Blueberry"},{"id":"1004","type":"Devil’s Food"}]},"topping":[{"id":"5001","type":"None"},{"id":"5002","type":"Glazed"},{"id":"5005","type":"Sugar"},{"id":"5007","type":"Powdered Sugar"},{"id":"5006","type":"Chocolate with Sprinkles"},{"id":"5003","type":"Chocolate"},{"id":"5004","type":"Maple"}]}

在日志中很难找到名称键，因为控制台上显示了很多无用的信息。当对象变的冗长时，难度更大。

我们使用 stringify函数 的第二个参数。我们重新编写代码并查看结果。

console.log(JSON.stringify(product,['name']);

{"name" : "Cake"}

问题解决了，与打印整个JSON对象不同，我们可以在第二个参数中将所需的键作为数组传递，可以只打印所需要的键。

# 2. 第二个参数（函数）

我们还可以将函数传入第二个参数进行传递。它根据函数中写入的逻辑计算每个键值对。如果返回 undefined ，则不会打印键值对。

请参阅此示例以获得更好的理解。

const user = {
 "name" : "Prateek Singh",
 "age" : 26
}

将函数作为第二个参数传入

{ "age" : 26 }

只打印了age，因为我们的函数条件返回字符串的值类型未定义。

# 3. 第三个参数（Number）

第三个参数控制最终字符串中的间距。如果参数是数字，则字符串化的每个级别都将以此数量的空格字符缩进。

// Note: '--' represnts the spacing for understanding purpose
JSON.stringify(user, null, 2);
// 结果
{
 "name": "Prateek Singh",
 "age": 26,
 "country": "India"
}

# 4. 第三个参数（String）

如果第三个参数是字符串，则将使用它代替上面显示的空格字符。

JSON.stringify(user, null,'**');
//{
//**"name": "Prateek Singh",
//**"age": 26,
//**"country": "India"
//}
Here * replace the space character.

# 5. toJSON 函数

我们有一个名为 toJSON 的方法，该方法可以作为任何对象的一部分，作为其属性。 JSON.stringify 返回此函数的结果并将其字符串化，而不是将整个对象转换为字符串。请参阅此示例。

const user = {
 firstName : "Prateek",
 lastName : "Singh",
 age : 26,
 toJSON() {
    return { 
      fullName: `${this.firstName} + ${this.lastName}`
    };
}
console.log(JSON.stringify(user));
// "{ "fullName" : "Prateek Singh"}"

我们可以看到，结果打印了 toJSON 函数的结果，而不是打印整个对象。
